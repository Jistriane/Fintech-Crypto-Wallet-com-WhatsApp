## Introdução

Bem-vindo à Fintech Crypto Wallet com WhatsApp! Nossa plataforma oferece uma solução completa para gerenciar seus ativos digitais com a conveniência de notificações via WhatsApp.

### Principais Funcionalidades

- Carteira multi-rede (Polygon, BSC, Arbitrum)
- KYC em níveis com limites progressivos
- Operações DeFi (Swaps, Liquidity Pools)
- Notificações via WhatsApp
- Segurança avançada

## Primeiros Passos

### Criando sua Conta

1. Baixe o aplicativo na App Store ou Google Play
2. Clique em "Criar Conta"
3. Preencha seus dados:
   - Email
   - Senha
   - Nome completo
4. Verifique seu email
5. Faça login no aplicativo

### Vinculando WhatsApp

1. Acesse "Configurações"
2. Clique em "Vincular WhatsApp"
3. Digite seu número
4. Insira o código recebido via WhatsApp
5. Configure suas preferências de notificação

## Carteira

### Visualizando seu Saldo

- A tela principal mostra seu saldo total em USD
- Lista de tokens com saldos individuais
- Gráfico de distribuição por rede
- Histórico de transações

### Enviando Tokens

1. Clique em "Enviar"
2. Selecione o token
3. Digite o endereço do destinatário
4. Insira o valor
5. Confirme a transação
6. Aguarde a notificação de confirmação

### Recebendo Tokens

1. Clique em "Receber"
2. Selecione o token
3. Compartilhe seu endereço ou QR code
4. Aguarde a notificação de recebimento

### Redes Suportadas

- Polygon (MATIC)
- BSC (BNB)
- Arbitrum (ETH)

## KYC

### Níveis e Limites

- **Nível 0**
  - Limite diário: $1,000
  - Limite mensal: $5,000
  - Sem verificação necessária

- **Nível 1**
  - Limite diário: $10,000
  - Limite mensal: $50,000
  - Documentos necessários:
    - RG/CNH (frente e verso)
    - Selfie

- **Nível 2**
  - Limite diário: $50,000
  - Limite mensal: $200,000
  - Documentos necessários:
    - Comprovante de residência

- **Nível 3**
  - Limite diário: $100,000
  - Limite mensal: $500,000
  - Documentos necessários:
    - Comprovante de renda

### Processo de Verificação

1. Acesse a seção "KYC"
2. Selecione o nível desejado
3. Envie os documentos necessários
4. Aguarde a análise
5. Receba notificação via WhatsApp

## DeFi

### Swaps

1. Acesse a seção "Swap"
2. Selecione os tokens
3. Digite o valor
4. Verifique a cotação e taxa
5. Confirme a transação
6. Aguarde a notificação de conclusão

### Liquidity Pools

#### Adicionando Liquidez

1. Acesse "Liquidity Pools"
2. Selecione a pool
3. Digite os valores
4. Confirme a transação
5. Receba seus LP tokens

#### Removendo Liquidez

1. Acesse "Minhas Posições"
2. Selecione a posição
3. Escolha o percentual
4. Confirme a remoção
5. Receba seus tokens

### Farming

1. Acesse "Farming"
2. Escolha uma pool
3. Deposite seus LP tokens
4. Acompanhe seus rendimentos
5. Colete as recompensas

## Segurança

### 2FA via WhatsApp

1. Ative em "Configurações > Segurança"
2. Receba códigos via WhatsApp
3. Use para:
   - Login
   - Transações
   - Alterações de configuração

### Dicas de Segurança

- Use senha forte
- Ative 2FA
- Nunca compartilhe códigos
- Verifique endereços
- Monitore notificações

### Recuperação de Conta

1. Clique em "Esqueci a senha"
2. Siga as instruções
3. Use o código via WhatsApp
4. Crie nova senha
